## 📝 TVING 클론코딩
##### 주제: TypeScript를 이용한 웹 사이트 만들기

----

## 🌏 주소
##### <https://vaiice.github.io/tving>

----

## 🕶 웹 기능

- TMDB API를 이용한 콘텐츠 제공
- 최신 영화 및 TV 프로그램 리스트
- 검색 및 상세 정보 확인
- 유튜브 예고편 바로 가기
